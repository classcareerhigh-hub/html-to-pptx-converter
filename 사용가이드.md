# 📋 PT 템플릿 사용 가이드

## 🎯 이 템플릿의 목적
이 HTML 파일은 **일관된 디자인의 PowerPoint**를 쉽게 만들기 위한 규격 양식입니다.
내용만 수정하면 → 변환기에 넣으면 → 예쁜 PPTX가 자동 생성됩니다.

---

## 📦 파일 구성
1. **presentation-template-20slides.html** - 20장 규격 템플릿
2. **html-to-pptx-converter.jsx** - 변환기 (Claude Artifact)
3. **사용가이드.md** - 이 문서

---

## 🚀 사용 방법

### STEP 1: HTML 파일 열기
- `presentation-template-20slides.html` 파일을 텍스트 에디터로 엽니다
  - VS Code 추천
  - 메모장도 가능 (단, UTF-8 인코딩 필수)

### STEP 2: 내용 수정
- 각 슬라이드는 `<section class="slide" data-page="XX">` 단위로 구분되어 있습니다
- 필요한 부분만 텍스트를 수정하세요
- 예시:
  ```html
  <h1>메인 타이틀을<br>입력하세요</h1>
  ```
  ↓ 수정
  ```html
  <h1>2025년 청년<br>취업 성과 보고</h1>
  ```

### STEP 3: 변환하기
1. Claude Artifact에서 변환기 열기
2. 수정한 HTML 코드 전체를 복사
3. 변환기에 붙여넣기
4. "PPTX 파일로 변환하기" 버튼 클릭
5. 자동으로 `.pptx` 파일 다운로드

## 📐 슬라이드 타입별 가이드

### 1️⃣ 표지 슬라이드 (01번)
```html
<section class="slide" data-page="01">
  <h2>여기에 상위 제목</h2>
  <h1>메인 타이틀을<br>입력하세요</h1>
  <p>
    서브 타이틀이나 간단한 설명을 여기에 작성합니다.<br>
    <span class="highlight">강조할 키워드</span>는 이렇게 표시됩니다.
  </p>
</section>
```

**수정 포인트:**
- `<h2>`: 작은 상위 제목 (예: "2025 프로그램 결과")
- `<h1>`: 큰 메인 제목 (예: "청년 취업 지원<br>성과 보고서")
- `<p>`: 부제 또는 설명
- `<span class="highlight">`: 파란색으로 강조할 단어

### 4️⃣ 제목 + 본문 슬라이드 (04번, 12번)

**수정 포인트:**

### 5️⃣ 리스트 슬라이드 (05번, 14번)
```html
<section class="slide" data-page="05">

  <h1>3가지 전략</h1>
  <ul>
    <li><strong>전략 1:</strong> 구체적인 설명을 여기에 작성</li>
    <li><strong>전략 2:</strong> 구체적인 설명을 여기에 작성</li>
    <li><strong>전략 3:</strong> 구체적인 설명을 여기에 작성</li>
```

**수정 포인트:**

- 항목 개수는 자유롭게 조정

### 6️⃣ 강조 박스 슬라이드 (06번, 16번)
```html
<section class="slide" data-page="06">
  <h2>핵심 메시지</h2>
  <h1>기억해야 할 것</h1>
  <div class="key-box">
    강조하고 싶은 핵심 문장을 여기에 작성합니다.<br>

  </div>
  <p>추가 설명이 필요하면 아래에 작성하세요.</p>
</section>
```

**수정 포인트:**
- `class="key-box"`: 파란색 테두리 박스
- 핵심 메시지 1-2문장 권장

### 7️⃣ 데이터 표 슬라이드 (07, 08, 13, 17번)

#### 3열 표 (07번)
```html
<table>
  <thead>
    <tr>
      <th>구분</th>
      <th>인원 (명)</th>
      <th>비율 (%)</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>1학년</td>
      <td>45</td>
      <td>28.8</td>
    </tr>
    <tr>
      <td>2학년</td>
      <td>62</td>
      <td>39.7</td>
    </tr>
  </tbody>
</table>
```

**수정 포인트:**
- `<th>`: 헤더 (파란 배경)
- `<td>`: 데이터 셀
- `<tr>`: 행 추가/삭제 가능

#### 4열 표 (08번, 13번)
- 열(column)이 4개인 표
- 사용법은 동일

## ✂️ 슬라이드 추가/삭제 방법

### 슬라이드 추가하기
1. 원하는 타입의 슬라이드 블록 전체 복사
   ```html
   <section class="slide" data-page="XX">
     ...전체 내용...
   </section>
   ```

2. 원하는 위치에 붙여넣기

3. `data-page="XX"` 번호를 순서대로 수정
   - 예: 21번, 22번, 23번...

### 슬라이드 삭제하기
- 해당 `<section>...</section>` 블록 전체 삭제
- 뒤의 슬라이드 번호는 자동으로 조정할 필요 없음

## 🎨 디자인 커스터마이징

### 브랜드 컬러 변경
파일 상단의 `:root` 부분에서 수정:
```css
:root {
  --color-primary:   #0071EC;  /* 파란색 */
  --color-secondary: #5050EF;  /* 보라색 */
  --color-accent:    #FF6B6B;  /* 빨간색 */
}
```

### 폰트 크기 조정
```css
h1 { font-size: 56px; }  /* 메인 제목 */
h2 { font-size: 20px; }  /* 상위 제목 */
p  { font-size: 24px; }  /* 본문 */
```

### 여백 조정
```css
--slide-padding: 80px;  /* 슬라이드 안쪽 여백 */
```

## ⚠️ 주의사항

### 1. 인코딩
- **반드시 UTF-8 인코딩**으로 저장하세요
- 안 그러면 한글이 깨집니다

### 2. 태그 닫기
- 모든 태그는 열면 닫아야 합니다
  ```html
  <h1>제목</h1>  ✅ 올바름
  <h1>제목       ❌ 태그 안 닫힘
  ```

### 3. 특수문자
- `<`, `>` 같은 문자는 `&lt;`, `&gt;`로 표시
- 따옴표는 `"` 또는 `'` 사용

### 4. 줄바꿈
- HTML에서 Enter는 무시됩니다
- 줄바꿈은 `<br>` 태그 사용

## 🔧 문제 해결

### Q1. 변환이 안 돼요
→ HTML 문법 오류 확인
→ `<section class="slide">` 태그가 제대로 닫혔는지 확인

### Q2. 한글이 깨져요
→ UTF-8 인코딩으로 저장했는지 확인

### Q3. 표가 이상하게 나와요
→ `<table>`, `<thead>`, `<tbody>`, `<tr>`, `<td>` 구조 확인

### Q4. 슬라이드가 20장보다 많아도 되나요?
→ 네! 원하는 만큼 추가 가능합니다

### Q5. 이미지를 넣고 싶어요
→ 현재 버전은 텍스트와 표만 지원합니다
→ 이미지가 필요하면 말씀해주세요 (업데이트 가능)
